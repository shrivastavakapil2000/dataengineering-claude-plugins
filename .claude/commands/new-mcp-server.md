Scaffold a new MCP server in this repository:
1. Ask for the server name and a brief description
2. Create a directory under `servers/<name>/`
3. Add a basic `README.md` with setup instructions
4. Add a starter `server.py` (Python) or `index.ts` (TypeScript) depending on user preference
5. Add a `requirements.txt` or `package.json` as appropriate
6. Update the root `CLAUDE.md` to list the new server
